# Simple Interest Calculator

This project provides a simple Bash script to calculate simple interest based on user-provided values.

## Description

The Simple Interest Calculator calculates simple interest using the following formula:

**Simple Interest = (Principal × Rate × Time) / 100**

Where:

* **Principal (P):** The initial amount of money
* **Rate (R):** The annual rate of interest
* **Time (T):** The time period in years

## Features

* Accepts principal amount as user input
* Accepts rate of interest as user input
* Accepts time period as user input
* Calculates simple interest
* Displays the calculated result

## Usage

Run the Bash script using:

```bash
bash simple-interest.sh
```

Enter the requested values for principal, rate of interest, and time period.

## Example

```text
Enter the principal:
1000

Enter the rate of interest:
5

Enter the time period:
2

The simple interest is:
100
```
